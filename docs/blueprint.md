# **App Name**: Ryzor AI MVP

## Core Features:

- PDF Upload and Storage: Allows users to upload PDFs and store them in Firebase Storage.
- PDF Processing: Triggers a Cloud Function to extract text from the PDF and split it into ~500-word chunks.
- AI Q&A: Offers a simple chat interface where users can ask questions and receive AI-generated answers based on retrieved chunks from the uploaded PDF, leveraging GPT-4o-mini as a tool.
- Chat Interface: The Chat Interface helps ask question based on uploaded files and retreive the relevant AI answer.

## Style Guidelines:

- Primary color: Dark electric blue (#7DF9FF) to signify digital power and innovation.
- Background color: Off-black (#121212), providing a clean dark theme to focus on content.
- Accent color: Electric Indigo (#6F00FF), highlighting interactivity and key UI elements.
- Body text: 'Inter' (sans-serif) for clean and readable UI and function descriptions.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-inspired feel in headings.
- Minimal, geometric icons for file uploads, chat, and AI actions.
- Two-page layout: one for drag-and-drop PDF upload, another for the chat interface.
- Subtle transitions on PDF upload and AI response display to enhance UX.
